feat: Add real-time audio format conversion support

Added comprehensive audio format conversion capabilities using Windows Media Foundation resampler:

Features:
- Real-time sample rate conversion (8000-192000 Hz)
- Channel conversion (1-8 channels, mono/stereo)
- Bit depth conversion (16/24/32 bits)
- High-quality resampling using Windows Media Foundation DSP
- Automatic format detection (PCM/IEEE Float)
- Buffer flushing on exit to prevent data loss

Implementation:
- Created AudioResampler class with proper resource management
- Integrated Windows Media Foundation Transform (MFT) API
- Added command-line options: --sample-rate, --channels, --bit-depth
- Implemented proper input/output buffer handling
- Added format validation and detailed error messages

Documentation:
- Updated README.md and README_CN.md with new features
- Added usage examples for speech recognition and music recording
- Included FFmpeg parameter mapping table
- Updated command-line options reference

Technical Details:
- Uses CLSID_CResamplerMediaObject for format conversion
- Handles WAVEFORMATEX and WAVEFORMATEXTENSIBLE formats
- Implements proper COM object lifecycle management
- Supports both event-driven and polling capture modes

Example Usage:
  # Speech recognition (16kHz mono 16-bit)
  wasapi_capture.exe --sample-rate 16000 --channels 1 --bit-depth 16

  # Music recording (44.1kHz stereo 16-bit)
  wasapi_capture.exe --sample-rate 44100 --channels 2 --bit-depth 16

Breaking Changes: None
Backward Compatibility: Fully maintained - all existing functionality works without changes
